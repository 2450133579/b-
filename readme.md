# 🌌 BiliBili AI 智能解析舱 (Bili-AI-Analyzer)

**极速提取视频语音 · GPU 硬件加速转录 · 万能大模型 AI 总结**

这是一个专为“省流”而生的本地化工具。它能帮你快速把 B 站长视频转化为精炼的文字总结，支持本地 GPU 加速，拒绝环境配置地狱，真正做到解压即用。

---

## ✨ 核心特性

- 🛠️ **真·免安装**：内置全套绿色 Python 环境（3.13.12），无需配置环境变量，不污染系统。
- 🧠 **内置大脑**：自带 Whisper `large-v3-turbo` 核心权重文件，断网也能进行高精度语音识别。
- 🚀 **显卡起飞**：深度适配 NVIDIA CUDA 加速，13 代 i7 + RTX 显卡秒出万字转录稿。
- 🌐 **万能适配**：兼容 OpenAI 格式的所有 API（DeepSeek, MiniMax, Claude, GPT-4o, Ollama 等）。
- 📂 **双重产出**：同时生成“纯净阅读版”和“时间轴字幕版”文本，方便快速回溯视频内容。

---

## 🚀 快速开始 (傻瓜三步走)

### 1. 解压即用
将下载的压缩包解压到一个 **全英文路径** 的文件夹下（建议放在磁盘根目录）。

### 2. 启动程序
双击根目录下的 **`一键启动.bat`**。
- 程序会自动检测本地环境与模型资产。
- 启动成功后，会自动在浏览器弹出操作界面。

### 3. 开始解析
在网页界面中：
- 粘贴 **B 站视频链接**。
- 填写你的 **API Key** 和 **接口地址**（用于 AI 总结）。
- 点击 **[ ✨ 启动智能流水线 ]**，剩下的交给 GPU。

---

## 📂 目录结构说明

- `python_env/` - 核心动力：内置的绿色 Python 环境。
- `model/` - 硅基大脑：Whisper 语音识别权重文件。
- `bilibili_works/` - 成果仓库：生成的转录稿和总结文件都在这里。
- `main.py` - 指挥中心：程序逻辑主文件。
- `cookies.txt` - 身份凭证：如需解析 1080P/大会员视频，请在此填入 B 站 Cookie。

---

## ⚠️ 注意事项

1. **硬件要求**：建议使用带有 NVIDIA 显卡的 Windows 设备以获得最佳体验（需安装 CUDA 驱动）。若无显卡，程序将自动降级为 CPU 模式。
2. **路径问题**：请确保项目路径中不包含中文或特殊字符，否则 FFmpeg 可能报错。
3. **API 配置**：总结功能需要联网。推荐使用 DeepSeek 等高性价比接口，接口地址务必包含 `/v1`（例如 `https://api.deepseek.com/v1`）。
